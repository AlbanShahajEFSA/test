TSEs monitoring data (aggregated level)

Aggregated level data should be specified prior detailing the different cases.

It is mandatory to fill in this information prior to be able to use the tool. The information can anyway be further edited and changed.

Creation of a monthly report

A new report can be created within the tool by clicking the File > New report button. This will open a new window, which allows selecting the needed report information, that are, the report year and month. Note that the report country is automatically filled with the one selected in the tool preferences (see TSEs preferences for further details).

Once the report information are filled in, the Create button of the window will be enabled. Press this button to create the new required report into your local machine. Note that the tool will now automatically check if the report already exists in DCF, therefore an internet connection is required to perform this step. In case of already existing report, the application will raise a warning, suggesting which actions are needed to solve the encountered issue.

At the end of this process, the new report is now created and can be opened (see Opening a monthly report for further details). In particular, a new report is stored in status Draft, meaning that you can edit the report contents.

Opening a monthly report

A report needs to be opened in order to edit or see its contents with the tool. In particular, this can be done by clicking the File > Open report button, which allows opening a report by selecting it from the list of all the local reports present in your machine. More precisely, once the desired report is selected, just press the Open button placed below the reports list to effectively open the report in the tool. Opening a report will display its aggregated data (if present) into the main table, and it will enable the user interface and its related buttons based on the report status.

Closing a monthly report

An opened report can be closed by clicking the File > Close report button, which is enabled only if a report is currently opened. This action will remove all the report information from the user interface.

Downloading a monthly report

It is possible that the data stored locally on your machine could be lost due to several reasons, such as crashes or hardware issues. For this reason, the tool allows downloading the reports you sent to DCF, in order to restore on your machine their last safe version.

A report can be downloaded by clicking the File > Download report button. However, note that this button is enabled only if no report is currently opened (see Closing a monthly report if you need to close the opened report). Pressing this button will open a list of all the reports you sent to DCF using the TSE tool, which allows selecting the report you want to download. Once the desired report is selected just press the Download button placed below the reports list to indeed start the download process. Note that if you already have the selected report in your machine, the tool will ask you a confirmation for overwriting it with the downloaded one, since the tool cannot store two different versions of the same report.

Once the report is successfully downloaded, it can be opened to see and possibly to edit its contents (see Opening a monthly report).

Sending a monthly report

A monthly report can be sent to DCF directly from the TSE tool without passing through the DCF website. This makes the send process much easier and, in particular, you can send the report just by pressing the contextual send button placed above the aggregated data table inside the Toolbar.
