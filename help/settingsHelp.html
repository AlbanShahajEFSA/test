Help

In each form of the tool, clicking on the icon it is possible to display an help explaining the data needed for the current table.

Initialization

At the first run you will be asked to fill in two forms:

Preferencies: Predefined default values to be used during the report editing. Predefined values are meant to ease the manual input of data (for recurrent information). The predefined values can be changed directly in the single records during the report editing (with the only exception of the "country" which is fixed in a same report and for the same data provider).
User settings: The information related to the user and used to perform the data submission to EFSA DCF. If you do not own a valid DCF account please send an email to Zoonoses Support.

It is mandatory to fill in this information prior to be able to use the tool. The information can anyway be further edited and changed.
