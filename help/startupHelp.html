TSEs report creator

The TSEs report creator tool allows the reporting of surveillance data on Bovine Spongiform Encephalopathy (BSE) in bovine animals, Scrapie and genotyping in small ruminants (sheep and goats) and Chronic Wasting Disease (CWD) in cervids. The tool allows the automatic submission of data into the EFSA Data Collection Framework (DCF). To be able to submit data to DCF you need to be registered as data provider for the TSEs data collection. If are not registered please send an email to Zoonoses Support. If your password expired please follow this link.

Content of a monthly report

Each monthly report shall contain all the data related to samples taken within a certain month (and supposed to be analyzed and evaluated within the same month).

Aggregated level data

Aggregated data (Number of positive cases, Number of negative samples, Number of inconclusive cases, Number of samples not suitable for testing) shall be summarized for a same combination of the following information:

Species: Animal species
Part: Sampled part of the animal
Prod: Type of production
Age: Age class
Target: Target group
Status: Status of the herd/flock

Case level data

For each POSITIVE and for each INCONCLUSIVE case, it will be required to specify the following information:

National case id: The univocal ID assigned at country level to the case
Index case: If the case is an index case, this have to be specified
Case assessment: The final evaluation of the sample (after discriminatory testing)
Animal id: The univocal ID assigned at country level to the animal
Herd id: The univocal ID assigned at country level to the herd/flock
Birth country: Country of birth of the animal

In addition, it will be required to detail the results of all the analysis run on the samples related to positive and inconclusive cases.

Random genotyping data

For small ruminants (sheep and goats), the tool allows also the reporting of random genotyping information.

Help

In each form of the tool, clicking on the icon it is possible to display an help explaining the data needed for the current form. The different pages of the help are also available from the help main page.

Initialization

At the first run you will be asked to fill in two forms:

Preferences: Predefined default values to be used during the report editing. Predefined values are meant to ease the manual input of data (for recurrent information). The predefined values can be changed directly in the single records during the report editing (with the only exception of the "country" which is fixed in a same report and for the same data provider).
User settings: The information related to the user and used to perform the data submission to EFSA DCF. If you do not own a valid DCF account please send an email to Zoonoses Support.

It is mandatory to fill in these two forms prior to be able to use the tool. The related information can anyway be further edited and changed.
